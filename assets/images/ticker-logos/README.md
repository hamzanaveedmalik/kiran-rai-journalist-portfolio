# Publication Logos

The following logo files are in this directory:

- `forbes.svg` - Forbes logo
- `gq.svg` - GQ logo
- `athlon-sports.svg` - Athlon Sports logo
- `sports-illustrated.svg` - Sports Illustrated logo
- `time-magazine.svg` - Times logo
- `daily-express.svg` - Daily Express logo
- `associated-press.svg` - Associated Press logo
- `khaleej-times.svg` - Khaleej Times logo

## Logo Format

- Format: SVG (Scalable Vector Graphics)
- SVG format provides crisp display at any size
- Logos are displayed with opacity effects and hover states
- If logos don't load, the publication name will be displayed as a fallback
